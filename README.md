# SkillBridge

A web-based micro-volunteering platform connecting skilled volunteers with short-term tasks.

## Tech Stack
- React.js
- Node.js
- Express.js
- MongoDB

## Setup

### Backend
cd server  
npm install  
npm run dev  

### Frontend
cd client  
npm install  
npm run dev  

## Features
- User authentication
- Task marketplace
- Task submission system
- Volunteer & Organization dashboards
